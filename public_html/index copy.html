How to add a page (three steps)

  1. Drop a file named /include/slug.main (slug: letters/digits/-/_).
  2. Add a nav item in /json/nav.json with {"label":"Name","href":"#slug"}.
  3. Reload and click the link — it loads the include into the page.

Notes (brief)

  • The loader injects HTML directly. Treat include files as trusted or sanitize them server-side.
  • If changes don't appear, hard-refresh (Ctrl/Cmd+Shift+R) or disable cache in DevTools.
